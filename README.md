고양이 키우기 게임

클릭과 방치를 통해 고양이를 성장시키는 방치형 육성 게임입니다.  
FastAPI 기반 백엔드 서버와 Python 클라이언트로 구성되어 있으며, Claude AI와 협업하여 개발했습니다.

게임 소개

닉네임과 고양이 이름을 등록하고, 클릭하거나 자리를 비운 사이에도 포인트가 쌓여 고양이가 성장합니다.

성장 단계 (총 10단계)
| 포인트 | 단계 |
|--------|------|
| 0 | 눈도 못 뜨는 아기 |
| 500 | 눈 뜬 아기 |
| 1,150 | 아장아장 |
| 1,900 | 호기심 많은 고양이 |
| 2,900 | 개구쟁이 |
| 4,000 | 의젓한 고양이 |
| 5,300 | 멋진 고양이 |
| 6,800 | 고양이 어른 |
| 8,500 | 현명한 고양이 |
| 10,400 | 전설의 고양이 |

기술 스택

- Python
- FastAPI
- SQLite3
- Pydantic
- Uvicorn

실행 방법

1. 패키지 설치
```bash
pip install fastapi uvicorn
```

서버 실행
```bash
python server.py
```

클라이언트 실행 (별도 터미널)
```bash
python client.py
```

API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/register` | 유저 등록 또는 재접속 |
| POST | `/click` | 클릭 포인트 획득 |
| GET | `/status/{nickname}` | 현재 상태 조회 |
